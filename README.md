## Interview Answers
**How to tell data stories?**
Data storytelling mein 3 cheezein hoti hain:
1. Context - problem kya hai
2. Insight - data kya keh raha hai
3. Action - kya karna chahiye
Charts aur visuals se complex data ko simple 
story mein convert karte hain.

**Executive reporting?**
Executive report short, clear aur action-oriented 
hoti hai. Technical details kam hote hain, 
business insights zyada hote hain. Key metrics, 
findings aur recommendations clearly present 
karte hain.
