## Interview Answers
**How to tell data stories?**
Data storytelling involves 3 elements:
1. Context - what is the problem
2. Insight - what does the data say
3. Action - what should be done
Charts and visuals convert complex data into a simple story.

**Executive reporting?**
Executive reports are short, clear, and action-oriented.
Less technical details, more business insights.
Key metrics, findings, and recommendations are presented clearly.
